# Quickstart: Get started with the plugin

The fastest way to use Attune AI is as a Claude Code plugin — 17
auto-triggering skills, no CLI, no API key.

**Install** (in Claude Code):

```
claude plugin marketplace add Smart-AI-Memory/attune-ai
claude plugin install attune-ai@attune-ai
```

**Then describe what you want** — the right skill activates:

```
review src/payments.py for quality issues and likely bugs
do a security audit on the src/ directory
what can attune do?
```

**Next:** Full walkthrough in the docs Getting Started → Quickstart (Plugin).
