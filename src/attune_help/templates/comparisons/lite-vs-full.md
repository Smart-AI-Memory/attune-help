# Comparison: Attune Lite vs Attune AI

Choosing between the lightweight plugin (no dependencies) and the full framework.

| Feature | Attune Lite | Attune AI |
| ------- | ------- | ------- |
| Install | No pip needed | pip install attune-ai |
| API key required | No | Yes (ANTHROPIC_API_KEY) |
| Skills | 5 | 13 |
| Workflows | 5 (via skills) | 17 (CLI + skills) |
| Wizards | 0 | 5 |
| Agent templates | 0 | 14 |
| MCP tools | 0 | 31 |
| CLI | No | Yes (attune command) |
| Tier routing | No | Yes (80-96% savings) |
| Telemetry | No | Yes (cost tracking) |

## Recommendation

Start with **Attune Lite** if you just want Claude Code skills with zero setup. Upgrade to **Attune AI** when you need CLI workflows, cost optimization, or the full MCP toolset.
