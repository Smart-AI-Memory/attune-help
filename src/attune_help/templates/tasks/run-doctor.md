# Task: Run environment health check

Diagnose configuration issues, missing dependencies, and environment problems. Run this when something isn't working as expected.

## Steps

1. **Run the health check**

   ```
   attune doctor
   ```

2. **Review findings**
   Doctor reports issues with severity levels and suggested fixes.

3. **Fix reported issues**
   Follow the suggested fixes for any WARN or ERROR items.
