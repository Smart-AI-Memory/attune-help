# Task: Manage lessons learned

Save, list, and remove lessons that persist across sessions. Lessons are stored locally and help avoid repeating mistakes.

## Steps

1. **Save a lesson**

   ```
   attune remember "Always use encoding=utf-8"
   ```

2. **List current lessons**

   ```
   attune lessons
   ```

3. **Remove a lesson**
   By line number or keyword.

   ```
   attune forget encoding
   ```
