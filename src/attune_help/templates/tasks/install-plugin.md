# Task: Install the attune-ai plugin for Claude Code

Install attune-ai as a Claude Code plugin to get slash commands, skills, and MCP tools directly in your Claude Code sessions.

## Prerequisites

- Claude Code CLI installed
- pip install attune-ai

## Steps

1. **Add the marketplace**

   ```
   claude plugin marketplace add Smart-AI-Memory/attune-ai
   ```

2. **Install the plugin**

   ```
   claude plugin install attune-ai@attune-ai
   ```

3. **Verify installation**
   Type /attune in a Claude Code session to test.
